# Custom Brave Browser Profile Avatars

Brave uses built-in avatar images for profiles. You can override these with custom images.

## Setup

Create the avatar override directory:

```bash
mkdir -p ~/Library/Application\ Support/BraveSoftware/Brave-Browser/Avatars
```

## Usage

1. Place your custom image in the `Avatars` folder with a matching filename
2. Restart Brave
3. Go to profile settings and select the avatar you replaced

Images should be square PNGs (recommended: 256x256 or 512x512).
